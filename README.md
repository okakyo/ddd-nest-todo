# NestJS x DDD

## Description

A TODO App with NestJS Fraamework by data-driven development.

## Installation

```bash
$ yarn install
```

## Requirements

- Docker Desktop
- nodenv


## Setup

1. Activate an docker.

```sh
docker compose up --build -d
```

2. Install Node.js
```sh
yarn install
```

3. run a api server
```sh
yarn dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
